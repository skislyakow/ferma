# Telegram Channel Factory — Универсальный фреймворк

Методология создания автоматизированных Telegram-каналов под любую нишу.
Основана на реальном опыте запуска @airdrop_crypto_ru.

---

## 1. Выбор ниши

### Критерии прибыльности (по приоритету):
1. **CPA** — есть ли офферы с высоким CPA ($10+)
2. **CPM** — платят ли рекламодатели за охват
3. **Объём контента** — есть ли 3+ канала-донора с ежедневным контентом
4. **Языковой барьер** — можно ли переводить с en → ru и получать уникальный контент
5. **Платёжеспособность аудитории** — готовы ли люди платить/кликать

### Матрица ниш:

| Ниша | CPA | CPM | Доноры | Перевод | Монетизация |
|------|-----|-----|--------|---------|-------------|

## 2. Поиск доноров

### Методика:
1. **Telegram search:** вбей ключевые слова ниши (crypto, finance, tech)
2. **TGStat / Telemetr:** смотри топ каналов по нише
3. **YouTube:** найди каналы по теме → в описании ссылки на Telegram
4. **Кросс-промо:** в крупных каналах смотри рекламу других каналов

### Чек-лист донора:
- [ ] Публичный (можно открыть t.me/s/CHANNEL)
- [ ] Есть текст + картинки (не только видео)
- [ ] Минимум 10 постов в день
- [ ] Англоязычный (чтобы перевод давал уникальность)
- [ ] Не спам/рекламный (иначе качество контента низкое)

### Проверка донора:
```
curl -s https://t.me/s/CHANNEL_NAME | grep -c "tgme_widget_message_wrap"
```
Если > 5 — канал парсится.

## 3. Технический сетап (копипаста)

### Структура:
```

├── .env
├── requirements.txt
├── run.py                 # Точка входа (копируй без изменений)
├── src/
│   ├── config.py          # Читает .env (копируй без изменений)
│   ├── db/
│   │   └── database.py    # SQLite (копируй без изменений)
│   ├── parser/

│   ├── publisher/
│   │   └── publisher.py   # Bot API + CPA (копируй без изменений)
│   └── filters/
```

### Что менять под новую нишу:
1. `.env` → SOURCE_CHANNELS, CPA_LINKS, язык
2. `filters.py` → AD_KEYWORDS под нишу
3. `publisher.py` → CPA_INSERT_EVERY (частота)
4. Создать нового бота у @BotFather
5. Создать новый канал

### Время запуска: 30 минут

## 4. Монетизация по типам

### A. CPA-партнёрки (самый доходный)

| Ниша | Партнёрка | CPA | Как вставлять |
|------|-----------|-----|---------------|
| Crypto | BingX, ByBit, Binance | $30-150 | В каждый 3-й пост |
| Finance | Mylead, Admitad | $10-100 | В каждый 2-й пост |
| Dating | CPAlead, Decibel | $5-50 | В каждый пост |
| Gaming | CPAlead, Admitad | $2-20 | В обзоры |
| Health | Admitad | $5-30 | В каждый 3-й пост |
| Tech | Admitad, прямые | $5-20 | В каждый 4-й пост |

### A2. CPA-партнёрки РФ и СНГ

Для русскоязычных каналов — **свой пул партнёрок**, принимающих карты РФ и работающих с СНГ-трафиком.

| Ниша | Партнёрка | CPA | Особенность |
|------|-----------|-----|-------------|
| Crypto | BingX, ByBit, Binance | $30-150 | Международные, работают с РФ |
| Finance | **Mylead** | $10-100 | Топ для РФ, фин. офферы |
| Finance | **Leadgid** | $5-80 | Кредиты, МФО, банки |
| Finance | **Kma.biz** | $5-50 | Микрозаймы, дебетовые карты |
| Dating | **Decibel** | $5-50 | Лучшая для РФ, платят в рублях |
| Dating | **Ad1.ru** | $3-30 | Dating + гемблинг |
| Gaming | **SalesDoubler** | $2-20 | Игры, приложения |
| Gambling | **Ad1.ru** | $10-100 | Казино, ставки (самые высокие CPA) |
| Health | **ActionPay** | $5-30 | БАДы, мед. услуги |
| E-commerce | **Admitad** | комиссия | Cashback, товарка |
| E-commerce | **CityAds** | комиссия | Товарные офферы |
| Education | **Mylead** | $5-50 | Курсы, вебинары |
| Подписки | **Admitad** | $2-10 | Сервисы, VPN, медиа |

**Как начать:**
1. Регистрируешься в Mylead / Leadgid / Kma.biz (принимают РФ карты)
2. Берёшь реферальную ссылку на оффер
3. Вставляешь в .env → CPA_LINKS
4. Получаешь выплаты на карту РФ

**Для новичка (рекомендую):**
- **Mylead** — самый простой вход, много офферов под любую нишу
- **Admitad** — международный, но выплаты в РФ работают через партнёров

### B. Telegram Ads
**Условие:** 1000+ подписчиков
**Доход:** $0.5-2 CPM (за 1000 показов)
**Подключение:** https://ads.telegram.org
**Ниши:** любые, но лучше Lifestyle, News, Memes (широкая аудитория)

### C. Прямые рекламодатели
**Когда:** 5000+ подписчиков
**Где искать:**
- Международные: @tgpromo, Telega.io, Serpstat
- **РФ / СНГ:** @tgpromo (русскоязычный), Telega.io (есть рублёвые выплаты), **@tgtop_bot**, **@crosspromo_bot**, **@telega_ads_chat**
- Биржи: **Telega.io** (выплаты на карты РФ через платёжных партнёров), **Serpstat** (есть рублёвые ставки)
- Чатики рекламодателей: **@tgpromo** (главный), **@tgtalk_chat**, **@telegram_ads**
- Прямые контакты: найди похожий канал → посмотри кто у них рекламируется → напиши тем же рекламодателям

**Цена:** $50-500 за пост (зависит от ниши и охвата)
**Как продавать:** кидай статистику канала (подписчики, охваты, ER)
**Выплаты:** РФ рекламодатели платят на карту/USDT/Crypto

### D. Продажа канала
**Когда:** канал набрал 10k+ подписчиков
**Где:** Telega.io, Telderi
**Цена:** 3-6 месячных доходов канала
**Пример:** канал с доходом $500/мес → продажа за $1500-3000

### E. Свои продукты / Инфобизнес
**Когда:** высокая лояльность аудитории
**Ниши:** Self-dev, Health, Finance, Cooking
**Как:** прогреваем → продаём курс/консультацию/бот

## 5. Стратегии масштабирования

### Стратегия A: "Сетка каналов" (рекомендуется)
1 канал → отладить до $200-500/мес
5 каналов → $1000-2500/мес
10 каналов → $2000-5000/мес

**Как делать:** одна кодовая база → разные .env → разные каналы

### Стратегия B: "Воронка"
- Канал-хаб (общая тематика)
- Узкие каналы (подниши)
- Перекрёстная реклама между ними

### Стратегия C: "География"
- Один контент → 3 языка (RU, EN, TR, etc.)
- Разные каналы на разные регионы
- Перевод через Yandex/OpenAI

## 6. Автоматизация всего цикла (чековер)

```
┌─────────────┐     ┌──────────────┐     ┌────────────┐     ┌─────────────┐
│ t.me/s/     │────▶│ Yandex       │────▶│ Фильтр     │────▶│ Bot API     │
│ (парсинг)   │     │ Translate    │     │ + Антиспам │     │ + CPA       │
└─────────────┘     └──────────────┘     └────────────┘     └─────────────┘
       │                    │                    │                   │
       ▼                    ▼                    ▼                   ▼
  Не требует            Дешево             Рейтинг по           Работает без
  API ключей            (бесплатно          просмотрам           Telegram API
                        1M симв/мес)       + блок рекламы
```

### Что НЕ нужно для запуска:
- ❌ Telegram API ID/Hash (my.telegram.org)
- ❌ DeepL, OpenAI (заменили Yandex)
- ❌ Свой сервер (работает с любого ПК)
- ❌ Админские права на донорских каналах
- ❌ Деньги на старте (кроме CPA на VPS $5-10)

### Что нужно обязательно:
- ✅ Telegram аккаунт (для создания бота и канала)
- ✅ Yandex Cloud API Key (бесплатно)
- ✅ VPS за $5-10/мес (для 24/7 работы)
- ✅ 30 минут внимания раз в неделю

### Подводные камни (из опыта):

| Проблема | Решение |
|----------|---------|
| `hash()` рандомизирован между запусками (PYTHONHASHSEED) | Используй `int(msg_id_str.split("/")[-1])`, не `hash()` |
| sendPhoto caption > 1024 символов → 400 ошибка | Если текст > 1024, пропускай картинку, шли sendMessage |
| t.me/s/ SSL ошибки из РФ | VPS за границей или SOCKS5 прокси |
| В донорских постах футеры ("Мы в VK", ссылки) | `_clean_footers()` вырезает строки с "мы в", "подпишись", URL |
| Посты-тизеры ("читайте на сайте") | `_is_external_source()` блокирует "читайте на", "источник:", "дайджест", "самые яркие" и т.д. |
| Один и тот же пост проверяется каждый цикл | `mark_skipped()` → published=-1, повторно не выбирается |

## 8. Типичные проблемы и решения

| Проблема | Решение |
|----------|---------|
| t.me/s/ не открывается | Включи VPN или используй VPS за границей |
| Нет новых постов у донора | Добавь ещё 2-3 донора |
| Бот не публикует | Проверь BOT_TOKEN и права админа |
| Низкое качество контента | Убери плохих доноров, добавь фильтры |
| CPA-ссылки не кликают | Поменяй формулировку, увеличь частоту |
| Канал не растёт | Купи 500-1000 ботов (Telebooster) для старта |
| Бан канала | Удали откровенный спам, уменьши частоту CPA |

## 10. Принцип

> Не ищи идеальную нишу. Ищи нишу, где есть 3 донора и 1 CPA-оффер.
> Запускай за 30 минут. Через месяц решай — убивать или масштабировать.

Рентабельность достигается объёмом: 1 канал = $100-500/мес при 15 минутах в неделю.
10 каналов = $1000-5000/мес при 2 часах в неделю.
